## Steps to setup te starter tempalte

1. Clone the project

```
git clone https://github.com/siddharthpalav/Express-Typescript-Starter-Project <ProjectName>
```

2. Move in to the folder structure

```
cd <ProjectName>
```

3. Make sure node version >= 24

```
nvm use 24
```

4. Install npm dependencies

```
npm i
```

5. Create a new .env file in the root directory and add the `PORT` env variable

```
echo PORT=3000 >> .env
```

6. Start the express server

```
npm run dev
```
